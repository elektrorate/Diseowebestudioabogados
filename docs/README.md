# Documentacion del Proyecto ONLEX

Este directorio centraliza el contexto tecnico y funcional del proyecto `Diseowebestudioabogados`.

## Estructura de documentacion

- `01-proyecto/`
  - `contexto.md`: objetivo, alcance y estado actual.
  - `stack.md`: stack, dependencias y decisiones tecnicas.
- `02-arquitectura/`
  - `estructura.md`: estructura de carpetas y capas.
  - `rutas-publicas.md`: mapa de rutas del sitio publico.
  - `rutas-admin.md`: mapa de rutas de administracion.
- `03-ui-estilos/`
  - `sistema-diseno.md`: colores, tipografia y reglas base.
  - `componentes-clave.md`: header, footer, formularios y componentes UI.
- `04-contenido-datos/`
  - `especialidades.md`: catalogo legal y estructura de datos.
  - `blog.md`: modelo de posts y categorias.
- `05-operacion/`
  - `desarrollo-build.md`: ejecucion local y build.
  - `despliegue.md`: configuracion para hosting (Vercel).
  - `mantenimiento-cambios.md`: cambios recientes y checklist de mantenimiento.

## Lectura sugerida

1. `01-proyecto/contexto.md`
2. `02-arquitectura/estructura.md`
3. `05-operacion/desarrollo-build.md`

## Nota

El contenido refleja el estado actual del repositorio en la rama `main`.
